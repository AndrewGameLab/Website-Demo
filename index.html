Tentang saya

Teknis, adaptif, dan selalu ingin tahu.

Palembang, Indonesia

Saya seorang game developer dan freelance programmer yang berfokus menciptakan game 2D/3D, mekanik menarik, serta custom tools yang membuat proses pengembangan lebih efisien.

Latar belakang Teknik Komputer dan Jaringan membentuk fondasi saya dalam troubleshooting sistem, jaringan, hardware, dan IT support. Dari sana, minat saya berkembang ke software engineering dan riset keamanan perangkat lunak.

Saya belajar melalui dokumentasi, komunitas, dan proyek mandiri. Saat ini saya terbuka untuk proyek freelance, dukungan infrastruktur IT, dan kolaborasi pengembangan game—baik secara individu maupun bersama tim.
